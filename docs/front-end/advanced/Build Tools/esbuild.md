# esbuild

## install

### npm

```shell
npm install -D esbuild
```

### yarn

```shell
yarn add -D esbuild
```



## usage

### npm

```shell
npx esbuild src.js --bundle --outfile=bundler.js 
```

### yarn

```shell
yarn esbuild src.js --bundle --outfile=bundler.js  
```

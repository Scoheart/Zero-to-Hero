# Rollup

## install

### npm

```shell
npm install -D rollup
```

### yarn

```shell
yarn add -D rollup
```



## usage

### npm

```shell
npx rollup src.js --file bundler.js --format es
```

### yarn

```shell
yarn rollup src.js --file bundler.js --format es
```

> rollup也只有打包作用
